# 高考优势
高考优势:
  - title: 低分录取
    icon: trophy
    content: 传媒专业录取分数线与音乐类专业相同。相比普通高考录取分降低100-200分给本科线以下的同学提供了一个上本科的机会

  - title: 简单易学
    icon: directions
    content: >-
      传媒专业属于艺术学科与人文学科结合的亚艺术门类。相比音乐、美术、舞蹈等纯艺术专业更加简单易学，
      时间与金钱的花费也相对较少。

  - title: 趣味性强
    icon: bubbles
    content: 传媒学习不同于文化课学习。传媒课堂主要以影视剧来辅助教学。趣味性大大增强。但趣味性强也要用功学哦。

  - title: 开设传媒专业的高校多
    icon: badge
    content: >-
      除北电、中戏等专门的传媒院校外，越来越多的综合类大学也增加了传媒专业。广东省内有暨大、
      华农、广大等知名高校都开设了传媒专业。

  - title: 就业前景广
    icon: rocket
    content: >-
      网络时代每个人都成为自媒体，传媒与网络的合流会激活一个巨量的传媒行业，诞生巨量的就业岗位。
      薪水也会随着工作经验的积累水涨船高。

  - title: 全面提升素质
    icon: globe
    content: >-
      传媒学习大大提升学生的综合素质。高中校园只重视学生的文化成绩。而传媒会提升语言表达、
      逻辑思维言谈举止等素质，会让孩子受益终身。

# 开设的课程内容
开设课程内容:
  - title: 编导类
    icon: lightbulb-o
    majors: >-
      电视编导、影视导演、文艺编导、影视文学 媒体创意、节目制作、艺术管理、文化管理、
      文化经济、公共事业、制片管理等
    content: >-
      命题演讲、视听语言分析、艺术概论、命题写作及技巧、编讲故事、即兴创作与评述、
      广播电视编导专业考试练习、语言表达、中外影视分析、影视基础知识等。
    career: >-
      编导、导演是各类电影电视节目的总指挥，该专业重点培养这方面的人才。一台节目的好坏，编导的能力在其中起到了
      很关键的作用，编导、导演是整个节目的核心。因此编导也具有核心的地位，有才华的编导一定会成为各个电视台争抢
      的焦点。这是一个充满创造性和挑战性的职业，虽然道路布满荆棘，但闪烁光环的前景仍然吸引了不少人投入其中。
  - title: 播音主持类
    icon: microphone
    majors: >-
      播音主持（双语）、播音主持（普粤）、国际交流、配音艺术等。
    content: >-
      普通话发音与发声、语言表达技巧、指定稿件播读与评述、文学作品朗读、主持人艺术技巧、
      节目策划与模拟主持、才艺展示、演讲与辩论等
    career: >-
      随着传媒和大众娱乐的不断发展，各地各级电视台之间的竞争也在不断激化，
      有能力、有个性的播音员和节目主持人成为了电视台、电台争夺的焦点。
      该专业文理兼收，毕业生能在广播电台、电视台及其他单位从事广播电视播音与节目主持工作。

  - title: 影视表演类
    icon: film
    majors: >-
      影视表演、话剧表演、时装表演、音乐剧表演、空乘等
    content: >-
      朗诵（作品选择, 朗诵技巧等）、
      声乐（演唱技巧, 发声方法等）、
      表演（小品表演, 即兴表演等）、
      形体（身体协调, 舞蹈等）
    career: >-
      随着传媒业的发展，媒体对表演类人才的需求量更是逐年剧增，表演毕业生就
      业较为灵活。文化产业，电影、电视剧、广告等事业为影视表演专业的人才提供了
      各种各样的岗位，本专业毕业生可以在影视剧制作中心、电视台、剧院或剧团、宣
      传文教部门、企事业单位和高校从事表演，宣传，教学等工作。
  - title: 摄影类
    icon: camera
    majors: >-
      数字媒体艺术、新媒体艺术、电影制作、剪辑、摄影摄像、图片摄影、照明艺术、
      电视摄像、商业摄影、数字电影技术等
    content: >-
      色彩基础知识、动漫技法、角色造型设计模拟现场拍摄、剧本创作、绘画基础、分镜头改编、
      影视后期处理、命题写作、摄影作品分析、电影图形处理、摄影基础知识、
      影视作品分析、艺术鉴赏等。
    career: >-
      高校开设摄影专业的还不多，但市场需求较大。上可进入艺术的象牙塔，下可
      丰衣足食受人尊重。摄影作为一个重要的艺术种类长期的被大众所关注并应用在诸
      多方面。例如商业摄影、人像摄影、教辅图书等方面，都离不开摄影。作为一名摄
      影师，以时尚、个性且富有内涵的形象气质与他们托起相机时的专注和潇洒，得到
      了越来越多人的尊敬和崇拜。
